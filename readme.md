
# this is my next-intl setup.


## Features


- Works for **App Router**.
- Is **dynamically routed** (`/en`, `/fr`, etc.).
- Makes **pages static**.
- **Server-side text loading** on first render for better SEO on client components.
- **No prop drilling**.
- Uses **latest next-intl version** (4.6.1).
- Is written with **Typescript**.
- Has setup **tailwind** to work out of the box.

## Getting Started

Navigate into the `version` directory and run:

```bash
npm install
```

then

```bash
npm run dev
```

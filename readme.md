
# this is my next-intl setup.


## Features

- Works for **App Router**.
- Is **dynamically routed** (`/en`, `/fr`, etc.).
- Makes **pages static**.
- **Server-side text loading** on first render, (even for client components) for better SEO.
- **No prop drilling**.

## Getting Started

Navigate into the `version` directory and run:

```bash
npm install
```

then

```bash
npm run dev
```

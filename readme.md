
# this is my working App Router next-intl setup.



## Features



- Preloads text in client components HTML for better SEO (allows HTML only bots to read text).
- Is **dynamically routed** (`/en`, `/fr`, etc.) for better SEO.
- **No prop drilling**.


- Variable and file names are more descriptive.
- Directory structure is better for larger project.

- Uses latest **next-intl** version (4.7.0).
- Uses latest **Typescript** version (5.9.3).
- Uses latest **Tailwind** version (4.1.18).




## Getting Started


```bash
cd project
```

```bash
npm install
```

```bash
npm run dev
```
